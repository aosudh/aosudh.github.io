第一篇文章：Hello, World

2026-06-18

这是我的第一篇文章 —— 博客终于上线了 🎉。

一直想有一个完全属于自己的角落，不依赖任何平台的排版和推荐算法，想写什么就写什么。于是有了这个小站：纯手写的 HTML 和 CSS，托管在 GitHub Pages 上，免费、简单、可控。

为什么不用现成的平台

  • 数据是自己的：所有内容都是仓库里的纯文本文件，随时备份、迁移。
  • 足够简单：没有数据库、没有后端，一个 HTML 文件就是一篇文章。
  • 可以慢慢长大：以后想加评论、加搜索、换静态生成器，都不迟。

接下来

我会陆陆续续把折腾过程中的笔记搬上来。如果你也想搭一个，可以看看下一篇《如何用纯 HTML 搭建并维护这个博客》。
