NVIDIA GPU PCI Device ID 全量对照：Ampere → Rubin

2026-06-24

一、为什么需要这张表

lspci -nn 只会告诉你 10de:2901 这种 vendor:device 组合——10de 是 NVIDIA，后面四位才是具体设备。但要从这四位 hex 反推"这是哪颗 die、属于哪代架构、对应哪款上市产品"，并不直观：同一颗 die 会派生出一堆 device ID（裸卡、HGX 模组、中国特供 SKU、CSP 内部变体、保留/未发布 ID），而且同一段 ID 空间里还混着板载 HD Audio、NVSwitch、Bridge/Root 等"不是 GPU"的功能。这张表就是把这些全部摊开、逐条对应。

数据来自 pci.ids（v2026.06.24） 与 NVIDIA 开源内核模块（open-gpu-kernel-modules）的 README 两个权威来源合并，再用 nouveau 代号、CUDA 文档与社区数据库（TechPowerUp / DeviceHunt 等）交叉核对（详见文末"数据来源与方法"）。总量分布：

  • Ampere 150 个（sm_80 / 86 / 87）
  • Ada Lovelace 67 个（sm_89）
  • Hopper 29 个（sm_90）
  • Blackwell 84 个（sm_100 / 103 / 110 / 120 / 121）
  • Rubin（下一代） 2 个（GR100 / GR102，sm_13x 待定）

二、怎么读这张表

每代一张表，列含义如下：

  • ID：device ID 的低四位 hex（完整写法是 10DE:<ID>）。
  • 芯片 / SM：物理 die 代号（如 GA102）与其 streaming-multiprocessor 架构等级；SM 与 Compute Capability 一一对应，速查见下表。
  • 功能：绝大多数是 GPU；加粗的是同一 ID 空间里的非 GPU 功能（HD Audio / NVSwitch / Bridge-Root）。
  • pci.ids 名称：来自 pci.ids 的完整名（含方括号变体），恒有值，是最稳的描述字段。
  • 产品名（官方 / 社区）：优先取 NVIDIA 开放内核模块 README 里的权威营销名；该列没有时退而取社区/第三方数据库的名字并标注"(社区)"（置信度低于官方）；都没有则为"—"。
  • 说明 / 推断：用于保留 / 内部 / 中国 SKU / 下一代等条目的背景与推断；[ES 工程样品] 表示工程样片 ID。
  • 开放驱动：✓ = 在 NVIDIA 开放内核模块的设备列表里；"—" = 只在 pci.ids、不在开放驱动列表（多为中国 SKU、数据中心变体、保留/下一代）。

八、数据来源与方法

这张表不是凭记忆拼的，而是两个权威来源合并、再用四类来源交叉核对的结果：

  1. pciutils / pci.ids（v2026.06.24）——超集，含新增的 Rubin GR100 / GR102；已确认与上游（ucw.cz 镜像）一致。
  2. NVIDIA/open-gpu-kernel-modules README——全部 831 条产品行（293 个 device ID、223 个名称，含 subsystem 级），提供官方名。
  3. torvalds/linux nouveau（nvkm）——代号交叉核对，结果吻合。
  4. NVIDIA CUDA 文档 + GTC 2026 + 路线图——每颗芯片的 SM / CC，以及 Rubin 的 GR 代号方案。
  5. TechPowerUp GPU DB + DeviceHunt + FreeBSD pci_vendors——为裸数据中心 / 中国 SKU 这类没有官方名的 ID 补社区名。
  6. 厂商产品页（NVIDIA、Lenovo 等）——确认 B100 / B200 / H 系列的产品命名。

检索结论：pci.ids + 开放驱动 README 两者合起来，就是完整的公开 device-ID 映射。外部源不再贡献新的 device-ID 级名称（除这里补的 18 个社区 ID）；剩余约 29 个裸 / 保留 ID 没有任何公开产品映射（内部 / 样片 / CSP / 下一代）。300 个 GPU 行全部有描述信息：271 个 pci.ids 方括号名、205 个 NVIDIA 官方名、18 个社区名、36 条推断说明，0 空白。

列约定：官方名取自 NVIDIA 开放内核模块 README（权威），"—" = 不在该列表；社区名来自 TechPowerUp / DeviceHunt / 厂商文档，置信度低于官方；说明 / 推断用于保留 / 内部 / Rubin 条目，基于路线图研究；"开放驱动" 的"—"表示在 pci.ids 但不在开放驱动列表。本表生成于 2026-06-24，基于当时的 main 分支。
